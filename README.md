# Programme Insights — User Documentation

Source for the Programme Insights user documentation, published with [Mintlify](https://mintlify.com).

- **Live docs:** https://docs.programmeinsights.co.uk *(once the custom domain is live)*
- **Config:** `docs.json` (navigation, theme)

Edits pushed to `master` deploy automatically via the Mintlify GitHub app.
